# myGame-coin-dropping-game

> Unity で作ったコイン落としゲーム

## 概要

Unity で制作した 3D コイン落としゲームです。
マウスクリックでコインを投入し、プッシャーでコインを押し出して落とすゲームです。

- プッシャーが前後に自動で動き、コインを落とす仕組み
- マウスの X 座標に応じてコインの投入位置が変わる
- コインには物理演算が適用されており、弾む・回転などがリアルに再現

## 使用技術

- Unity（C#）
- Unity Physics（Rigidbody / PhysicMaterial）

## 動作環境 / 注意事項

- Unity でプロジェクトを開いて実行してください
- 動作確認済みの Unity バージョンは不明です（古いバージョン向けの書き方が含まれている可能性があります）
